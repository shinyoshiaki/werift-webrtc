# Werift WebRTC DataChannel ベンチマーク

このパッケージはWerift WebRTCのDataChannelに対するベンチマークテストを提供します。

## 機能

以下の3種類のベンチマークが利用可能です：

1. **シンプルテスト** (`simple`): 基本的なRTT（往復時間）測定。1KBのデータを使用した単純なエコーテスト。
2. **サイズテスト** (`size`): 異なるサイズのデータ（1KB, 10KB, 100KB, 1MB）を使用したパフォーマンス測定。
3. **オプションテスト** (`options`): DataChannelの各種設定（順序保証、再送回数、パケット生存時間）による性能の違いを評価。

## セットアップ

必要な依存関係をインストールします：

```sh
npm install
```

## 実行方法

### シンプルテスト

```sh
npm run simple
```

基本的なDataChannelのRTT（往復時間）を測定します。1KBのデータを100回エコーしてRTTを計測します。

### サイズテスト

```sh
npm run size
```

複数のデータサイズを使用してパフォーマンスを評価します：
- 小サイズ (1KB)
- 中サイズ (10KB)
- 大サイズ (100KB)
- 超大サイズ (1MB)

### オプションテスト

```sh
npm run options
```

DataChannelの各種設定を変更してパフォーマンスを比較します：
- デフォルト設定（順序保証あり）
- 順序保証なし
- 再送回数制限
- パケット生存時間制限
- 順序保証なし＋再送なし

## 結果の出力

各ベンチマークでは以下の情報が出力されます：

- 平均RTT（往復時間）
- 最小RTT
- 最大RTT
- 中央値RTT
- スループット（Mbps）
- エラー数（該当する場合）
- パケットロス（該当する場合）

結果はテーブル形式とCSV形式の両方で出力されます。
