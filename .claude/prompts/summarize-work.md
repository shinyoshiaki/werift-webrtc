## 作業サマライズのプロンプト

このプロンプトを使用して現在の作業セッションをサマライズし、次回のセッションで継続できるようにします。

```
作業内容を次のclaude codeセッションで再開できるようにサマライズしてください
```

その後、以下のコマンドを使ってサマリーをファイルに保存します：

```
サマリを.claude/summary/${title}.mdに保存してください
```

`${title}` には、作業内容を表す簡潔なタイトル（例：ice-manager-implementation）を指定してください。

### サマリーに含めるべき情報

1. **実施した変更**
   - 作成/修正したファイル
   - 実装した機能や変更した処理
   - 主要なクラス・メソッドの役割

2. **将来の計画**
   - 今後実装予定のコンポーネント
   - 拡張予定の機能

3. **次のステップ**
   - 直近で取り組むべきタスク
   - 考慮すべき課題や解決方法

4. **リファクタリングの効果**
   - コード品質や保守性への影響
   - パフォーマンスや拡張性への影響
